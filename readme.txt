WP Page Links allows you to choose a number of page links to display.

== Description ==

The WP Page Links Plugin will allow you to add page links to your Sidebar.

What you can customize:

1. Give the plugin a title.
2. Choose how many page links you want to display on the page.  There is a limit of 20 links.
3. How do you want the pages to be sorted in the list?  You can choose Date Posted, Date Modified,...etc.
4. How do you want the pages to be listed? Highest to Lowest or Lowest to Highest?
5. If you want to exclude a page just enter the page ID.
6. Show your love and add a link to our plugin site.

== Installation ==

1. Upload wp-page-links folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins menu in the Dashboard.
3. Drag the 'WP Page Links' widget to either your Sidebar or Footer
